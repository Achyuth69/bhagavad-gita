# Bhagavad Gita Chapters Page

This folder contains the chapters browsing interface for the Bhagavad Gita project.

## Files:

- **chapters.html** - Main chapters page displaying all 18 chapters
- **chapters.css** - Complete styling with animations and responsive design
- **chapters.js** - Interactive functionality and chapter data
- **chakra.png** - Sudarshana Chakra background image (copied from BV folder)

## Features:

✨ **Visual Design:**
- Rotating Sudarshana Chakra background
- Galaxy/cosmic background with particle effects
- Golden accent colors matching the project theme
- Divine custom cursor with glow effect

🎨 **Chapter Cards:**
- All 18 chapters with Sanskrit names and English translations
- Verse counts for each chapter
- Elegant hover effects with golden glow
- Staggered fade-in animations
- Responsive grid layout

🔄 **Interactive Elements:**
- Sticky navbar with scroll effects
- Clickable cards (currently shows alert, can be extended for individual chapter pages)
- Chakra spins faster on card hover
- Smooth transition animations

## Navigation:

- **Home** → `../DASHBOARD/index.html`
- **About** → `../DASHBOARD/index.html#about`
- **Chapters** → `chapters.html` (current page)
- **TeamXplore** → `../DASHBOARD/index.html#team`
- **Contact** → `../DASHBOARD/index.html#contact`
- **Translate** → `../BV/translation.html`

## Usage:

1. Open `chapters.html` in a web browser
2. Browse through the 18 chapters
3. Click on any chapter card (currently shows alert for future development)
4. Use navigation to access other parts of the application

## Dependencies:

- Google Fonts: Cinzel & MedievalSharp
- Background images from BV folder
- Dashboard CSS for unified navbar styling
